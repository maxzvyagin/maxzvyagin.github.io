CrossedWires Python Package

Library created to explore the divergence between semantically equivalent models in PyTorch and TensorFlow and their relationship to the hyperparameter space using three different computer vision models and the CIFAR10 dataset. Open dataset with call for community contributions to improve model equivalence across deep learning frameworks.
